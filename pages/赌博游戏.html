凯利公式 — 完整原理、数学推导 & 硬币游戏落地应用

一、场景与变量定义（适配当前硬币游戏）

本游戏为重复独立博弈：反复抛硬币，每次仓位按当前总资产固定比例下注。

  • p：获胜概率（正面） = 0.5
  • q：失败概率（反面） = \(1-p = 0.5\)
  • b：净赔率 = 2（下注1份，赢了额外赚2份，总资产变为 \(1+2f\)）
  • f：下注比例（待求最优仓位，\(0 \le f \le 1\)）
  • 赢后资产系数：\(1 + b f\)
  • 输后资产系数：\(1 - f\)（输掉本次下注金额）

二、凯利公式 完整数学推导过程

凯利准则目标：最大化长期资本几何平均增长率。

步骤1：定义单轮资产增长因子

单次游戏后，资产的期望对数收益率（几何增长用对数求解更简便）：

G(f) = \(p \cdot \ln(1+bf) + q \cdot \ln(1-f)\)

G(f)：每一轮的对数增长率；最大化 G(f) 等价于最大化长期总资产。

步骤2：对 f 求导，找极值点

连续可导函数最大值出现在一阶导数 = 0 处：

\(G'(f) = \dfrac{pb}{1+bf} - \dfrac{q}{1-f} = 0\)
步骤3：解方程化简

令导数等于 0，移项：

\(\dfrac{pb}{1+bf} = \dfrac{q}{1-f}\)

交叉相乘：

\(pb(1-f) = q(1+bf)\)
步骤4：展开并整理含 f 项
\(pb - pbf = q + qbf\)
\(pb - q = pbf + qbf\)
\(pb - q = bf(p+q)\)
步骤5：利用概率性质 \(p+q=1\) 最终化简

因为获胜+失败概率总和为1：\(p+q=1\)

\(pb - q = bf\)

解出最优下注比例 \(f^*\)：

\(\boldsymbol{f^* = \dfrac{bp - q}{b}}\)

✅ 至此，标准凯利公式推导完成。

三、代入硬币游戏参数，算出理论最优仓位

已知：\(p=0.5,\ q=0.5,\ b=2\)

\(f^* = \dfrac{bp - q}{b} = \dfrac{2 \times 0.5 - 0.5}{2} = \dfrac{1 - 0.5}{2} = \dfrac{0.5}{2} = 0.25\)

结论：本游戏理论最优仓位 = 25%，和页面模拟结果完全对应。

四、简化场景验证（一赢一亏）

取最典型场景：一轮赢、一轮亏，两轮后总资产：

\(F(f) = (1+2f)(1-f) = 1 + f - 2f^2\)

这是开口向下二次函数，求最大值：求导 \(F'(f)=1-4f=0\)

\(1-4f = 0 \implies f=0.25\)

再次证明：25% 是局部最大值仓位。

七、实战应用提醒（股票/投资场景）

  • 凯利公式仅适用于正期望、可重复博弈；负期望游戏最优解是：不参与。
  • 现实投资中胜率、赔率无法精确预知，建议使用半凯利（f*/2）降低回撤。
  • 仓位超过凯利最优值后，长期收益必然下降，风险指数上升。
  • 核心思想：控制仓位 = 控制寿命，活下来才能享受复利。
